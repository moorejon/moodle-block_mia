This is a proof of concept block for Moodle Instructional Assistant built against Moodle version 2.9

Advice is roughly mapped to Moodle Pedagogy progression https://docs.moodle.org/29/en/Pedagogy#Progression but only
implements basic advice for the first 5 levels to illistrate the concept of a smart assistant to help teachers improve
course design.

Installation
Install as other Moodle blocks by copying to the moodle_path/blocks folder
